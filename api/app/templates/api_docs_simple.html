Complete reference for rowing conditions API endpoints

https://api.samwduncan.com

Quick Reference

GET
https://api.samwduncan.com/api/complete
Get all current data, forecasts, and scores in one comprehensive response
GET
https://api.samwduncan.com/api/rowcast
Get current RowCast score and all parameters used in the calculation
GET
https://api.samwduncan.com/api/rowcast/forecast
Get RowCast scores for the next 24 hours with detailed conditions
GET
https://api.samwduncan.com/api/rowcast/forecast/extended
Get extended RowCast forecast (up to 7 days) using NOAA data when available
GET
https://api.samwduncan.com/api/rowcast/forecast/2h
Get RowCast score for 2 hours from now (supports: 30m, 1h, 2h, 1d, etc.)
GET
https://api.samwduncan.com/api/weather/current
Get current weather conditions including temperature, wind, and UV index
GET
https://api.samwduncan.com/api/water/current
Get current water conditions including flow rate, temperature, and gauge height
GET
https://api.samwduncan.com/api/noaa/stageflow
Get NOAA NWPS stageflow data including current observations and forecasts

RowCast Scoring

Score Range: 0-10 (higher is better)

  • 8.0-10.0: Excellent rowing conditions
  • 6.0-7.9: Good conditions with minor considerations
  • 4.0-5.9: Fair conditions - exercise caution
  • 0.0-3.9: Poor conditions - not recommended

Factors considered:

  • Temperature (optimal: 74-85¬∞F)
  • Wind speed and gusts
  • Water flow rate and temperature
  • Precipitation and UV index
  • Safety alerts and weather warnings
